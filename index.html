Quadruped Spider Robot

A four-legged walking robot with 3 servos per leg (12 total), controlled by a Raspberry Pi Pico and a PWM driver via I²C, while also making use of ToF and IMU sensors. Implements multiple gaits: forward/backward locomotion, rotation, balance on a moving platform, and height adjustment to clear obstacles.

Using an OOP approach, the development of this robot allowed for a more modular architecture. Each leg's trajectories were computed offline following Bezier Curves for better performance, while the Inverse Kinematics are computed in real-time.

Simplified Perception System for Autonomous Boat

A simplified perception system ready for deployment on an autonomous boat. Detects obstacles using a LiDAR sensor and builds a persistent occupancy grid map in real time, updated incrementally. Efficient enough for embedded hardware, and capable of distinguishing static from dynamic obstacles by applying a fine-tunned decay rate to each cell of the grid — essentially removing LiDAR measurement noise and updating the map.

Computer Vision · Sports Analysis

Basketball Shot Detection System

A computer vision system for automatic shot detection. Assunming a fixed camera setup, after user-based manual point and line selection on an initial image, the system automatically estimates the 2D Ground Plane Homography using the RANSAC algorithm, as well as the camera's calibration parameters.

Using a YOLOv8 object detection model — trained with a custom dataset — the system is capable of detecting the 2D position of the basketball throughout the video, and tracks it frame-by-frame.

By performing partial 3D Reconstruction (assuming the ball always lies on a vertical shooting plane), the system can determine whether the ball passed through the hoop.

Control Systems · Embedded · MATLAB

Ball Balancer — Self-Balancing Beam

A wooden beam that autonomously balances a ball on top of it using a single servo motor. A closed-loop PID controller continuously reads ball position and adjusts the beam angle to keep the ball centred. Demonstrates core principles of classical control applied to a physical plant.

The PID Controller was designed and simulated on MATLAB assuming a simple second-order linear model of the open-loop system.
